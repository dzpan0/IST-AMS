# Overview

This project was developed as part of the Systems Analysis and Modeling (AMS) course at IST-UL.
The objective is to design a conceptual architecture for the AgriRetail Alliance, a collaboration between farmers and retailers aiming to coordinate agricultural supply and demand through a shared platform.
